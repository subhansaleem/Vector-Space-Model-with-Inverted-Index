This is a Vector Space Model Web App designed on Python Web Framework DJANGO.


To run this app:
	1.make a virtual environment by "py -m venv <project-name>"
	2.activate the virtual environment.
	3.run this command to install all the supported libraries: "pip install -r requirements.txt"
	4.go to the directory with manage.py file.
	5.run the following command to run the server: "python manage.py runserver"
	6.go to localhost to check the app.

Thankyou!
