This is a Vector Space Model designed on Python with few libraries like 're', 'nltk' etc.

To run this app:
	1.make a virtual environment by "py -m venv <project-name>"
	2.activate the virtual environment.
	3.run this command to install all the supported libraries: "pip install -r requirements.txt"
	4.go to the directory with main.py file in main dircetory.
	5.run the following command to run the app: "py main.py"
